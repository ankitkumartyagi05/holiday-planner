## ğŸ§  Logic Overview

The application uses a state-based architecture pattern:

1. **State Object:** Holds current country, currency, selected destination, and budget totals.
2. **Router:** A custom JS function toggles `display: none/block` on HTML sections to simulate navigation.
3. **Currency Logic:**
    * Calculates base costs in USD.
    * If `country === 'India'`, multiplier `* 83` is applied dynamically.
